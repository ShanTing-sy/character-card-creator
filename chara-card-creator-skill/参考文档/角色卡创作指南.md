# 角色与世界创作原则

## 角色创作核心原则

### 原则一：Show, Don't Tell（展示而非讲述）

角色的一切特质必须通过具体、可观测的外部细节来展现，禁止直接陈述内心活动或性格标签。

**NG 示范（直接解释）**：
- "他是一个很勇敢的人。"
- "她因为童年经历而对人不信任。"
- "他这么做是为了复仇。"

**OK 示范（外部呈现）**：
- "他踏前一步，把发抖的少年挡在身后。喉咙滚了一下，但声音没颤。"
- "她接咖啡时指尖离杯壁始终隔着半寸——从不碰到别人碰过的地方。"
- "他把那枚铜扣在指间翻来覆去地搓了三年，边缘的字已经看不清了。"

**实操检查清单**：
- [ ] 删除所有"因为...所以..."式的因果解释句
- [ ] 用身体语言替代情绪形容词
- [ ] 用对话细节（音量、语速、停顿、措辞）替代性格陈述

### 原则二：限知视角

所有描写必须严格锚定在 {{user}} 的第一人称感知范围内。仅可描写 {{user}} 看到、听到、触摸到、闻到、感知到、思考到的内容。NPC 的内心想法、隐藏动机禁止直接陈述，仅可通过其外在行为间接暗示。

**绝对禁止**：
- "他在心里盘算着如何背叛你"（{{user}} 不可能知道）
- "她其实很担心你，只是不说"（上帝视角）
- 切换到 NPC 所在场景描写 {{user}} 不在场的事

**允许**：
- "他笑了一下，但嘴角的肌肉绷得太紧，眼角却没跟着动"
- "她说了句'没事'，转过去看窗外。玻璃上映出的脸模糊，看不清表情"

### 原则三：五感落地

所有场景描写必须落地到具象的五感细节。禁止空泛化表述。

| 空泛表述 | 具象改写 |
|---------|---------|
| "很安静" | "只有日光灯管在头顶嗡鸣——那种老化了以后才会发出的、断断续续的电流声" |
| "很难受" | "胃里像塞了块湿冷的抹布，喉咙口涌上一股酸" |
| "很漂亮" | "夕阳刚好斜过窗沿，落在那只陶碗的缺口上，镀了层薄薄的金" |
| "看不清" | "走廊尽头的光太暗了，只能看见一团模糊的影子，像是站着，又像是吊着" |

感官权重按场景分配：
- **日常场景**：放大嗅觉/视觉的烟火气
- **恐怖场景**：放大听觉/触觉的细微异常（水滴声、温差、气流）
- **逃亡场景**：放大呼吸/心跳的体感、脚下的触感

### 原则四：对白差异化

每个角色说话的方式应像指纹一样独特。从以下维度设计：

1. **词汇选择**：惯用哪些词？绝对不用哪些词？
2. **句子长度**：短促有力还是绵长婉转？
3. **口头禅与语气词**：哪些话是"只有这个角色才会这么说的"？
4. **话语中的隐藏信息**：话里的潜台词是什么？哪些真话在当假话说，哪些假话在当真话说？

**练习方法**：将不同角色放在同一场景下对同一事件的反应写出来。如果三句话看不出是谁说的，说明对白差异化不够。

### 原则五：能力与性格的共鸣

角色的能力/馈赠应当是性格的延伸，而非随机附加的"外挂"。好的能力设计满足：

- **性格→能力**：能力效果直接反映角色的核心价值观或最深的执念
- **经历→能力**：能力的来源与角色的关键人生事件关联
- **代价→成长**：能力的使用有代价，这个代价为角色成长提供动力

**正确示例**：
- 一个总是被要求"快点"的人，觉醒了时间减速能力——不是因为他想要力量，而是因为他渴望片刻的喘息
- 一个习惯隐藏痛苦的人，获得了"储存伤害然后奉还"的能力——因为这就是他一直在做的事

**错误示例**：
- "他获得了控制火焰的能力，因为火很酷"
- "她是全能法师，什么都会"

### 原则六：群像角色互补法则

多角色设计必须确保每个角色有独特的存在价值：

**性格光谱覆盖**：
- 积极性 ↔ 消极性（乐观/悲观、主动/被动）
- 外向性 ↔ 内向性（社交/孤僻、表达/沉默）
- 理性 ↔ 感性（逻辑驱动/情感驱动）
- 规整 ↔ 随性（计划控/即兴派）
- 利他 ↔ 利己（奉献/算计）

**能力互补矩阵**：
- 确保每类能力有主担当和副担当
- 主攻/副攻/辅助/控制/治疗各维度至少覆盖 3 个
- 避免两个角色能力高度重叠

**关系张力网络**：
- 每个角色对至少与其他 3 个角色有明确的情感/利益关联
- 关联类型多样化：信任/猜疑、依靠/竞争、崇拜/轻视、亲密/疏远
- 画出关键的情感三角或对立轴

## 世界创作核心原则

### 世界观的一致性检验

以下问题可以帮助检验世界观是否自洽：

1. **物理法则**：这个世界有什么是和现实不同的？这些不同带来了什么连锁影响？
2. **权力结构**：谁掌握资源？他们为什么能掌握？这个结构稳定吗？裂缝在哪？
3. **日常细节**：这个世界的人早餐吃什么？他们最担心什么？他们的笑话是什么样的？
4. **信息传播**：普通人如何获取信息？他们对世界的了解有多少是真、多少是假？

### 世界书编写时机

不必一开始就写完所有世界书。以下时间点最适合新增条目：

1. **开局前**：核心铁则（2-3 条常驻条目）
2. **入场时**：角色所在的场景描述
3. **对话中**：每次 AI 遗忘/误解时，立即补一条
4. **转场时**：新场景的地点、氛围、规矩
5. **揭示时**：隐藏设定的逐步放出

### 世界观的比例感

一个令人沉浸的世界不需要解释一切。黄金比例：
- **20% 已写出的**：写进世界书的明确设定
- **30% 暗示的**：通过描写间接触及的，给读者脑补空间
- **50% 留白的**：你自己知道但没写出来的，让世界有"呼吸的空间"

## 文风快速适配表

| 场景类型 | 节奏 | 句式 | 感官权重 | 禁忌 |
|---------|------|------|---------|------|
| 安全区日常 | 舒缓松弛 | 短句为主，口语化 | 嗅觉>视觉>听觉 | 紧绷压抑、空洞宏大 |
| 中式恐怖 | 慢→快渐进 | 日常铺垫→短促碎裂 | 听觉>触觉>视觉 | 直白血腥 jump scare |
| 日式恐怖 | 极慢积累 | 清冷克制绵长 | 听觉>触觉 | 喧闹、过度解释 |
| 规则怪谈 | 逻辑→崩坏 | 公文式短句→矛盾破碎 | 认知>视觉 | 情感泛滥 |
| 竞技逃亡 | 极快 | 短句+强动词 | 体感>听觉 | 冗余描写、多愁善感 |
| 生存挑战 | 写实粗粝 | 白描为主 | 触觉>体感>嗅觉 | 理想化、悬浮 |
| 玄幻/仙侠 | 大气→紧张 | 中式美学长句 | 视觉>体感 | 西方化用语 |
| 科幻 | 精准冷静 | 数据感+规则感 | 视觉>听觉 | 语感软化 |
| 童话改编 | 前期柔→后期碎 | 柔软→诡异渐变 | 视觉>听觉 | 全程黑暗 |

## 质量自检清单

### 角色维度
- [ ] 该角色的性格能从 3 个以上行为细节中体现（而非直接陈述）？
- [ ] 该角色的说话方式是否独特到可以"盲听识人"？
- [ ] 该角色的能力是否与性格/经历有因果关系？
- [ ] 该角色是否有明确的欲望、恐惧和底线？
- [ ] 该角色在群像中是否有不可替代的价值？

### 世界维度
- [ ] 世界的核心铁则是否已写入世界书（constant: true）？
- [ ] 经济和资源如何流通是否有逻辑闭环？
- [ ] 主要场景是否都有具体的五感描写？
- [ ] 各势力的关系是否有内在张力（不是铁板一块）？
- [ ] 是否有留白让故事自然生长？

### 文风维度
- [ ] 是否始终保持限知视角（无上帝视角）？
- [ ] 是否删除了所有"因为...所以..."式心理分析句？
- [ ] 场景描写是否有具体的五感细节（不空泛）？
- [ ] 不同场景是否应用了不同的文风适配？
- [ ] 日常场景是否保持轻松松弛的基调？
