# ⬡ 图床工具

一个轻量、极简的文件转 URL 链接工具。上传文件到 GitHub 仓库，通过 jsDelivr 全球 CDN 加速访问，所有平台均可正常显示。

## 文件说明

| 文件 | 适用场景 | 布局 |
|------|---------|------|
| `index.html` | **桌面端**（推荐） | 左右布局：左侧上传区 + 右侧列表 |
| `mobile.html` | **移动端 / 手机** | 上下布局：上传区在上 + 列表在下 |

## 特点

- ☁️ **免费** — 基于 GitHub 仓库 + jsDelivr CDN，零成本
- ⚡ **极速** — 全球 CDN 加速，加载飞快
- 🔒 **安全** — 随机哈希命名，无法被猜测访问
- 📄 **全格式支持** — 任何文件类型均可上传（单文件 ≤ 25MB）
- 🗄 **存储库** — 方块网格浏览，支持标签分类和搜索筛选
- 🏷 **标签系统** — 上传后可添加标签，按标签筛选
- 💾 **配置模板** — 可保存多套配置（多仓库/多 Token）一键切换
- 🎨 **极简设计** — 黑白灰简约风格

## 使用前配置

1. 拥有一个 **GitHub 仓库**
2. 打开页面，点击右上角 **✱** 查看详细教程
3. 按教程生成 GitHub Token（点击 **⚙ 设置** → 填入 Token 和仓库链接）
4. 点击 **💾 配置模板** 可保存多套配置方便切换

> **仓库链接格式**：直接粘贴 GitHub 上目标文件夹的完整链接
> 例如：`https://github.com/用户名/仓库名/tree/main/images`
> 工具会自动解析仓库名、分支和路径。

## 使用方法

| 方式 | 操作 |
|------|------|
| **拖拽上传** | 将文件拖入上传区域 |
| **点击上传** | 点击 **「📤 上传文件」** 按钮选择文件 |
| **文件夹上传** | 点击 **「📂 文件夹」** 按钮选择整个文件夹 |

## 完整功能流程

```
上传 → 已上传列表 → ✤ 存入存储库（可选）→ 🗄 存储库浏览
                                        ↓
                                  🏷 添加标签 → 🔍 搜索/筛选 → 📋 复制链接
```

### 存储库功能

- 点击 **🗄 存储库** 按钮打开独立弹窗
- 方块网格布局展示所有已存文件
- 🔍 搜索框按文件名或标签搜索
- 🏷 标签栏点击筛选，小按钮样式
- 📋 复制链接 / 🏷 编辑标签 / ✕ 删除
- 「清空当前」只删除当前筛选条件下的文件

### 按钮说明（桌面版）

| 按钮 | 位置 | 功能 |
|------|------|------|
| 📤 上传文件 | 左侧 | 打开文件选择器 |
| ✕ | 上传队列 | 取消排队中的文件 |
| ✤ | 已上传列表 | 存入存储库（可选标签） |
| 📋 | 已上传/存储库 | 复制 CDN 链接 |
| 💾 | 右上角 | 配置模板管理（保存/加载多套配置） |
| ✱ | 右上角 | 使用教程与声明 |
| ⚙ | 右上角 | 设置（Token、仓库链接） |
| 🗄 | 已上传右侧 | 打开存储库弹窗 |

## 链接格式

```
https://cdn.jsdelivr.net/gh/用户名/仓库名/路径/文件名
```

## 注意事项

- 单文件最大 **25MB**
- 支持 **所有文件格式**
- Token 仅存储在浏览器本地 localStorage
- 历史记录最多保存 200 条
- 存储库最多保存 500 项
- GitHub API 每小时 5000 次请求限制
- 请勿上传非法、侵权或敏感内容
