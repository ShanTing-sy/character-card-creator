# 正则表达式模式目录

本文档收录正则助手支持的通用正则模式，作为生成正则的参考。

## JS 正则引擎注意事项

SillyTavern 的正则在 JavaScript 引擎上执行，有以下限制：

- **后行断言** `(?<=...)`：ES2018+ 支持，但部分旧环境可能不兼容，优先考虑替代方案
- **先行断言** `(?=...)`：广泛支持
- **dotAll 模式** `s` 标志：使 `.` 匹配换行符，ES2018+
- **非贪婪匹配** `*?` 和 `+?`：广泛支持
- **Unicode 模式** `u` 标志：支持中文等多字节字符匹配

## 类别1：内容清除模式

### 单标签清除
```
/<标签名>[\s\S]*?<\/标签名>/gi
```
- `[\s\S]*?` 匹配任意字符（含换行），非贪婪
- `gi` 全局 + 大小写不敏感

### 多标签联合清除
```
/<tag1>[\s\S]*?<\/tag1>|<tag2>[\s\S]*?<\/tag2>|<tag3>[\s\S]*?<\/tag3>/gi
```
- 用 `|` 连接多个标签匹配
- 可在前面加 `\s?` 清除标签前的空白

### HTML 片段清除
```
/<!--[\s\S]*?-->|<p\s+style[^>]*>|<\/p>/gi
```
- HTML 注释：`<!--[\s\S]*?-->`
- 带属性的开标签：`<tag\s+attr[^>]*>`
- 闭标签：`<\/tag>`

### 特殊符号清除
```
/<@>|<#>|<\/?extra>/gi
```
- 占位符
- 可选的开闭标签 `<\/?tag>`

## 类别2：深度过滤模式

### 框架内关键词过滤
```
/(?<=<框架标签>[\s\S]*?)目标关键词[\s\S]*?(?=<\/框架标签>)/i
```
- `(?<=<框架标签>[\s\S]*?)`：在框架开标签之后
- `(?=<\/框架标签>)`：在框架闭标签之前
- 用于在特定框架内定位并删除指定内容

### 变体：仅保留摘要
```
/^[\s\S]*?(<摘要标签>[\s\S]*?<\/摘要标签>)[\s\S]*$/i
```
- 捕获摘要部分 `$1`
- 替换为 `$1` 实现"只发送摘要"

## 类别3：内容隐藏模式

```
/<标签名>[\s\S]*?<\/标签名>/g
```
- 仅 `g` 标志，不需要 `i`（标签名通常大小写固定）
- 替换为空字符串

## 类别4：用户输入处理模式

### 全文捕获
```
^([\s\S]*)$
```
- 捕获整个文本为一个组
- 替换字符串中使用 `$1` 引用

### 包裹输入
替换字符串示例：
```
以下是用户的本轮最新输入：\n<user_input>\n$1\n</user_input>
```

### 清除历史输入
- 正则同上，替换为空
- 配合 `minDepth: 1` 只清除非首轮

## 类别5：美化注入模式

### 标题解析（7字段标准格式）
```
/<title>\s*字段1：\s*(.*?)\s*\|\s*字段2：\s*(.*?)\s*\|\s*...\s*<\/title>/s
```
- `s` 标志允许 `.` 匹配换行
- `(.*?)` 非贪婪捕获各字段
- `\s*` 容忍空白变化
- 替换字符串中使用 `$1` 到 `$7` 引用捕获内容

### 替换字符串结构
```html
<link rel="stylesheet" href="外部CSS">
<style>自定义CSS规则</style>
<div class="容器">
  <!-- HTML 结构，引用 $1, $2, ... -->
</div>
<style>附加CSS规则和动画</style>
```
- CSS 使用 `currentColor` 适配深色/浅色主题
- 动画使用 CSS `@keyframes`
- 必须包含 `@media (max-width: 500px)` 移动端适配

## 类别6：样式统一模式

```
/<标签名>/g
```
替换为：
```
<标签名 style="display: block; text-align: center; opacity: 0.6; cursor: pointer;">
```

## 类别7：关键词过滤模式

```
/关键词1|关键词2|关键词3/g
```
- 用 `|` 连接多个关键词
- 替换为空字符串
- 注意转义正则特殊字符（如 `.` → `\\.`）

## 常见转义对照表

| 原始字符 | JSON 中的正则 | 说明 |
|----------|-------------|------|
| `\s` | `\\s` | 空白字符 |
| `\S` | `\\S` | 非空白字符 |
| `\.` | `\\.` | 字面点号 |
| `\|` | `\\|` | 字面竖线 |
| `\/` | `\\/` | 正则结束符转义 |
| `\n` | `\\n` | 换行符 |
