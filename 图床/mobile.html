✱ 使用教程

📖 步骤
⚠️ 声明

📦 配置 Token

  1. 打开 github.com/settings/tokens
  2. 点 "Generate new token" → "Tokens (classic)"
  3. 取个名字，勾选 repo
  4. 生成并复制 ghp_... 开头的 Token
  5. 点 ⚙ 设置 → 粘贴 Token → 保存

📤 上传文件

  • 拖拽到上方区域，或点「上传文件」/「文件夹」按钮
  • 支持所有格式，单文件 ≤ 25MB

🔗 复制链接

  • 在「已上传」列表中点击 📋 复制 CDN 链接

⚠️ 声明

🚨 重要
本工具仅为个人使用设计。

1. 数据安全
Token 仅存 localStorage，仅发往 api.github.com。

2. 文件存储
文件存入你的 GitHub 仓库。随机文件名防止猜测。

3. API 限制
每小时 5000 次请求，单文件 ≤ 25MB。

4. CDN 缓存
新文件需几分钟全球生效。

5. 责任
请勿上传违法内容，后果自负。

✅ 总结
随机文件名保护是核心安全机制。不传链接即安全。
